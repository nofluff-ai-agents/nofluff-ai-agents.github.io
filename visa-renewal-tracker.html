Prompt 1 — Upload CSV for instant renewal schedule
Copy and use with your CSV upload
I have uploaded a CSV file containing employee visa and Emirates ID data. Each row includes: employee name, nationality, department, visa expiry date, EID expiry date, and an individual alert buffer (in days) for each employee. Please note — the alert buffer varies per employee and should be respected individually: - Some employees have a 7-day buffer (simple renewals, UAE nationals with ICP app access) - Others have a 15, 30, 45, or 60-day buffer (complex visa types, overseas processing, visa changes) - Use each employee's specific buffer to determine whether action is required NOW — not just whether the document has expired Please analyse the data and provide: 1. A summary count — how many employees are: expired, within their alert buffer (critical), approaching within 60 days (upcoming), and safe 2. A prioritised renewal schedule sorted by urgency — most critical first, showing: - Employee name and nationality - Visa expiry date and EID expiry date - Days remaining until expiry - Their individual alert buffer - Whether they are currently within their buffer period - Recommended action and suggested start date for renewal 3. Flag any employees where both visa and EID expire on the same date — these can often be processed together to save time and cost 4. Flag any employees where the visa and EID expiry dates differ significantly — these may need separate processing timelines 5. A concise this-week action list — only the employees requiring immediate action Format as a clean, structured report the PRO team or HR manager can act on immediately.

Colour codes the Google Sheet rows automatically: red for expired/within buffer, amber for upcoming 60 days, green for safe 6. Does NOT send an email if there are zero employees requiring attention that week Please write the complete script with clear comments explaining each section, and provide step-by-step instructions for how to paste it into Google Sheets and set the weekly trigger.
